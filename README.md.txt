# Projeto Long Sword (ARPG)

## Visão Geral

Este é um projeto de desenvolvimento de um ARPG single-player com inspirações em jogos como Path of Exile e Hades. O objetivo é criar uma experiência de jogo completa, desde o protótipo até o lançamento em plataformas como Steam e Epic Games.

Este repositório acompanha o progresso de um roteiro de desenvolvimento detalhado.

## Principais Milestones

**Protótipo Jogável (0-2 meses):** Mecânicas principais, 1 mapa, 1 personagem, 3 skills, 4 inimigos e sistema de loot.
**Alpha Técnico (2-4 meses):** Gameplay completo de um ato, sistema de save/load e menu principal.
**Beta de Conteúdo (4-8 meses):** Múltiplos atos, mais inimigos, mais skills e sistema de quests.
**Lançamento (11-12 meses):** Build final, arte implementada e publicação.
